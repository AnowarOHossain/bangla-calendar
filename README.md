# 🌼 Bangla Calendar

A **simple**, **beautiful**, and **responsive** web application that displays the **Bangla calendar** with dates, months, seasons, and real-time clock — all in Bangla numerals and language. It accurately converts Gregorian dates to Bangla dates and presents the data in a clean, user-friendly interface.

🔗 **Live Demo**: [Bangla Calendar](https://banglacalendar.netlify.app/)


## ✨ Features

* 📅 **Bangla Calendar View** – Clean monthly calendar in Bangla with numerals and month names.
* 🔄 **Gregorian to Bangla Conversion** – Converts Gregorian dates accurately based on the Bangla calendar system.
* 📌 **Today's Info** – Displays today's date, day, month, and season in Bangla.
* 🕰️ **Real-Time Clock** – Stylish digital clock showing Dhaka time, updated every second.
* 📱 **Responsive Design** – Seamlessly works across mobile, tablet, and desktop.
* 💫 **Smooth Animations** – Slide transitions while switching between months.
* 🎨 **Modern UI** – Tailwind CSS and custom styles deliver a crisp and elegant look.

## ⚙️ How It Works

* Converts Gregorian dates to Bangla using **14 April 594 AD** as the reference starting point.
* Displays a **dual-calendar view**: Bangla and Gregorian dates side-by-side.
* "Today" section shows real-time **Bangla date, day name, month, and season**.
* "Time" section shows current time in **Asia/Dhaka** timezone.
* Includes **navigation buttons** to switch months with slide-in animations.
* The UI is **fully responsive** and adjusts beautifully across devices.

## 🚀 Getting Started

### 1. Clone the Repository

bash
git clone https://github.com/anowarohossain/bangla-calendar.git

### 2. Navigate to Project Directory

bash  
cd bangla-calendar

### 3. Open the App

* Directly open `index.html` in your browser
  *or*
* Use a local development server:

bash
  npm install -g live-server
  live-server
  

> ⚠️ No external build tools required. Tailwind CSS is included via CDN, and fonts are loaded from Google Fonts.
